# symmetrical-adventure

A simple program for generating QR Codes quickly and easily.

---

## 🚀 Features

* Generate QR Codes instantly
* Simple and lightweight
* Easy to use
* Customizable input data

---

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/Aditya-5harma/symmetrical-adventure.git
cd symmetrical-adventure
```

Install dependencies (if any):

```bash
pip install qrcode[pil]
```

---

## ▶️ Usage

Run the program:

```bash
python qr_code.py
```

Enter the text or URL you want to convert into a QR code.

The QR code will be generated and saved in the project directory.

---

## 📸 Example

Input:

```
https://github.com
```

Output:

* A generated QR code image file
